Mini logos de bancos usados no módulo Financeiro (Caixas e Bancos).

- Objetivo: manter identidade visual rápida no cadastro e nos cards de contas.
- Regra atual: prioridade para cor oficial + mini logo circular.
- Fonte: logos baixadas da internet (domínio oficial de cada banco, com fallback de favicon),
  salvas localmente para uso offline e renderização rápida.
