# TDLib — binários nativos (Gestão YAHWEH)

Baixe com:

```bat
cd C:\gestao_yahweh_premium_final
.\scripts\download_tdlib.ps1
```

Ou só Android:

```bat
.\scripts\download_tdlib.ps1 -AndroidOnly
```

Versão alinhada ao pacote `libtdjson` ^0.3.0: **TDLib 1.8.65** (up9cloud).

Pastas geradas:
- `android/app/src/main/jniLibs/*/libtdjson.so`
- `ios/Frameworks/libtdjson.xcframework`

Credenciais: `lib/features/chat/data/tdlib_credentials.dart`
